Thesis Demo Setup

Using VM
  • make sure that the IP is 192.168.56.101 or perform the steps from below that involve changing the IP
  • go to https://192.168.56.101
New Installation
  • there are 2 servers (todo-app, website)
  • install nodejs/npm (todo-app), optionally with forever
  • make sure python3 is available (website)
  • install charm for python3 (build from source, required building pbc lib as well)
  • make sure apache is enabled with ssl
  • add apache rules for ssl virtual host to the two server (see below)
  • import schema.sql (requires mysql server)
  • make sure the mysql server has a user root with pwd root
  • replace CERT_HASH in tSokeServer.py with the server's cert hash
  • install mongodb for the todo-app
  • edit iframe url according to the used url/ip in client-app.html
  • edit event.origin in main.js of the todo-app according to url/ip
  • allow mixed content in the browser (the todo app is currently served via http)
Apache vhost config SetEnv proxy-sendcl ProxyPreserveHost On ProxyPass / http://127.0.0.1:8080/ ProxyPassReverse / http://127.0.0.1:8080/
