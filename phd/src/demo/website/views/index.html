Cryptographic Password Framework

This is the BPR admin website. Go ahead and define a password policy. When you're done head over to the client registration website to register a new user.

Password Policy

The regular expression specifies mandatory character sets and their number.

  • d = ASCII digits
  • u = ASCII upper case letters
  • l = ASCII lower case letters
  • s = ASCII symbols
The default policy requires one character of each subset and at least 5 characters in total.

About

This is a demo to demonstrate the concept of Blind Password Registration and according password-based login using a tag-based PAKE protocol to resemble a PACCE protocol.
